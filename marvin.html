Projects

Bounce Game

The player's goal is to move a ball across the screen to reach a token while avoiding a series of obstacles.

  • Developed this application with Python and the Pygame library.
  • Designed with an object oriented programming model.
  • Utilized Class instances and methods to help keep track of game object interaction.
  • Handled user input keyboard events and updated the game window accordingly.

Wordle

This is a Python Shell implementation of the popular online game Wordle where a player has 6 chances to guess a mystery 5 letter word.

  • Performed Opening, reading, writing and closing files.
  • Decrypted a corrupted file with a text of words used in the Wordle dictionary.
  • Searching and sorting correct or incorrect letters used in guesses.
  • Operates on python shell.
